# Weather API
This is project based on the Roadmap.sh challenge. The API fetchs the [Visual Crossing's API](https://www.visualcrossing.com/weather-api/), and return weather conditions data of the locations. This project uses:

- FastAPI
- Docker
- Redis to cache weather data (server-side) 

# Runing the project

You need to run the following command inside the project dir:

``` CMD
docker compose up
```

After that, you can access the following url:

- [Project url](http://0.0.0.0:8000)
